📸 BANNER IMAGE FOR INLINE BOT
================================

This folder should contain a banner image for the inline bot protection system.

Required file: banner.jpg
Location: core/assets/img/banner.jpg

Recommended specifications:
- Format: JPEG (.jpg)
- Dimensions: 1280x720 or 1920x1080 (16:9 ratio)
- Max file size: 5 MB
- Content: Herikku Userbot logo/branding

The banner will be shown to unauthorized users who try to use your inline bot.

If the banner.jpg file is not found, the system will fall back to text-only message.

To add your banner:
1. Create or download a banner image
2. Save it as "banner.jpg" in the core/assets/img/ folder
3. Restart the bot

Example banner content ideas:
- Herikku logo with gradient background
- "Private Userbot" text with modern design
- GitHub repository link overlay
- Minimalist tech-themed design
